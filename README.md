# Weather Data Analysis and Heatwave Detection (Rajasthan, India 2006–2025)

## Project Objective

The objective of this project is to analyze historical weather data from Rajasthan, India, between 2006 and 2025. The analysis focuses on identifying temperature trends, detecting heatwave events, and understanding seasonal weather patterns through data visualization and statistical analysis.

The project demonstrates the use of Python for data cleaning, feature engineering, exploratory data analysis (EDA), and visualization of climate-related data.

**Dataset Name:** Heatwave Dataset Rajasthan India (2006–2025)

### Dataset Features:

* YEAR
* MONTH
* DAY
* TEMP2M (Temperature at 2 meters)
* TMAX (Maximum Temperature)
* TMIN (Minimum Temperature)
* DEW2M (Dew Point Temperature)
* RAIN (Rainfall)
* CLOUD (Cloud Cover)
* WIND_U10, WIND_V10 (Wind Components)
* MSLP (Mean Sea Level Pressure)
* SRAD (Solar Radiation)
* EVAP (Evaporation)
* SOIL Variables
* HEATWAVE Indicator

The dataset contains daily weather observations collected over a 20-year period and provides valuable information for climate and heatwave analysis.

## Key Findings

### Temperature Trends

* Summer months recorded the highest average temperatures.
* Winter months showed significantly lower temperature levels.
* Temperature patterns followed clear seasonal cycles.

### Heatwave Analysis

* Multiple years experienced a high number of heatwave days.
* Heatwave frequency varied across the study period.
* Extreme temperature events were concentrated during the summer season.

### Weather Relationships

* Strong relationships were observed between maximum and minimum temperatures.
* Weather variables such as solar radiation and cloud cover influenced temperature variations.

### Seasonal Patterns

* Summer exhibited the highest temperatures and heatwave occurrences.
* Monsoon months showed increased rainfall and lower temperatures.
* Winter remained the coolest season throughout the study period.
