# Rajasthan Heatwave Analysis (2006–2025)

## Project Overview
This project analyzes heatwave and weather patterns in Rajasthan from 2006 to 2025 using Python.

## Dataset
Source: Kaggle Rajasthan Heatwave Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Visualization
5. Heatwave Analysis
